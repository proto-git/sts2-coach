# STS2 Coach — knowledge base

This folder is the brain. The system prompt re-reads every `.md` file in this
directory on each advice call, so you can edit these freely without restarting
the app.

File conventions (numeric prefix controls ordering when concatenated):

- `00-overview.md`    — this file
- `10-characters.md`  — character-specific win conditions
- `20-archetypes.md`  — common deck shapes per character
- `30-relics.md`      — relic evaluations
- `40-cards.md`       — card evaluations in context
- `50-acts-bosses.md` — act/boss priorities and prep thresholds
- `60-shop.md`        — shop theory, gold thresholds, remove-vs-buy
- `70-events.md`      — event decision trees

The coach research pass (run periodically as the meta shifts) updates these files.
