# Shop theory (placeholder)

Default priorities when entering a shop:
1. Card remove if gold allows and deck has obvious dead weight.
2. A relic that advances the current plan (not a generic "good" relic).
3. Potions that cover known boss pain points.
4. Cards last, and only if they materially change the deck's ceiling.

Skip everything if the run is already winning and the pickups are marginal.
